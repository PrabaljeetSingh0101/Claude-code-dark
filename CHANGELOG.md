# Changelog

All notable changes to the "Claude Code Dark" theme will be documented in this file.

## [1.1.0] - 2026-07-19

### Changed
- **Editorial UI Restructuring:** Enforced the "Invisible Chrome" design principle by making structural UI borders (panels, peek views, dropdowns) completely transparent, allowing the layout to breathe.
- **Active Spotlights:** Replaced the "tech blue" (`#6A9BCC`) active border lines with a warm, parchment-tinted silver (`#C3C2B7`) to indicate active tabs and panels without causing visual fatigue.
- **Text Warmth:** Shifted the primary foreground text color from a cool, clinical white (`#EAECF0`) to a warm, printed-paper off-white (`#E7E6E1`) to emulate physical ink and paper.
- **Notification Elevation:** Updated the VS Code notification toast background to the official Claude Android "Snackbar" color (`#323232`), instantly creating depth and elevation against the editor background.
- **Strategic Notification Badges:** Changed the Activity Bar notification badge from blue to the True Claude Terracotta Orange (`#D77757`), acting as a highly visible, strategic spotlight against the quiet UI.
- **Terminal Polish:** Aligned the terminal cursor color to match the official 2026 Dark default (`#bfbfbf`).
