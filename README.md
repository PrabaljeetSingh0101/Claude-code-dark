# Claude Nine Theme

A meticulously crafted dark theme for VS Code, directly inspired by and extracted from the official **Claude Code CLI** binaries. 

Built to feel premium, focused, and authentically Anthropic.

## ✨ Features

- **Authentic Colors:** The editor surface, brand accents (`#D77757`), and success/error/warning states are 1:1 matched to Anthropic's own terminal palette.
- **Glassmorphism Hints:** Subtle 85% opacity in hovers, autocomplete widgets, and the Command Palette to let your syntax gently bleed through.
- **Perfected Contrast:** A beautiful, eye-friendly visual hierarchy where the editor serves as the spotlight (`#1F1F1E`), wrapped in deep, distraction-free chrome (`#1A1A19` and `#181817`).
- **Premium Polish:** Features italicized comments, bold foundational keywords, and completely unified UI elements (Peek View, Merge Conflicts, Git Decorations, and Debuggers).

## 💡 Philosophy

Be kind, write beautiful code, and let your tools do justice to your creativity. This theme was built from the ground up to get out of your way and let your code shine.
